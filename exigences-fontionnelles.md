#  Exigences Fonctionnelles et Non Fonctionnelles

Une liste des exigences fonctionnelles et non fonctionnelles pour le projet **Community**: :
## Ressources

| 🛠️ Fonctionnelles                | 🔐 Non fonctionnelles                 |
| ------------------------------- | ------------------------------------ |
|  **Partage de ressources(lien, description, etc.)**  |  **Bonne lisibilité des messages (embed, emojis, formatage)**        |
|  **Recherches de ressources**   |  **Réponse rapide aux commandes**  |
|  **Système de tags**            |    **Gestion des doublons**                  |


## Entraides
 
| 🛠️ Fonctionnelles                | 🔐 Non fonctionnelles                 |
| ------------------------------- | ------------------------------------ |
|  **Système d'entraide**         |**Cloturation automatique du sujet**   |
|  **Demande d'aide**             |**Indexation des réponses**           |
|  **Système de validation de réponse**| **Anti-abus (limiter le spam)**  |
|  **Système de vote de réponse** |                                      |
|  **Recherche des questions**       |                                      |

## Profil
 
| 🛠️ Fonctionnelles                | 🔐 Non fonctionnelles                 |
| ------------------------------- | ------------------------------------ |
|  **Création de profil**         |  **Protection des données personnelles** |
|  **Demande de profil d'un utilisateur**             | **Respect du RGPD (suppression sur demande)**         |
|  **Recherche de profil**|                                 |

## Gamingfication

| 🛠️ Fonctionnelles                | 🔐 Non fonctionnelles                 |
| ------------------------------- | ------------------------------------ |
|  **Gagner des points pour chaque contribution (ressource, réponse validée, etc.)**         | **Classements mis à jour régulièrement**  |
|  **Commande pour voir son classement ou celui des autres**             |    **Notifications discrètes (pas de spam dans les canaux)**      |
|  **Recherche de profil**|                                 |
