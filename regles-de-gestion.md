# Règles de gestion

## Ressource
**RG 01** : Une ressource doit avoir:
- Un titre
- Une description
- Une date de publication
- Un auteur
**RG 03**: Une ressource peut avoir des étiquettes.
**RG 04**: Une ressource peut recevoir des notes positives et négatives.
**RG 05**: Une ressource peut avoir des commentaires.
**RG 06**: Une ressource peut être épinglée par un modérateur.
**RG 07**: Une ressource doit référencer son message d'origine même le message d'origine n'est plus disponible.
**RG 08**: Une ressource doit référencer un lien (médias compris)
**RG 09**: Une ressource peut être ajoutée par tout membre
**RG 10**: Une ressource peut être supprimée par son auteur ou un modérateur
**RG 11**: Le corps d'une ressource peut être mise à jour uniquement par son auteur


## Recherche
**RG 12**: Une recherche peut être triée par date.
**RG 13**: Une recherche doit avoir des termes.
**RG 14**: Une recherche peut référencer les champs
**RG 15**: Une recherche peut trier par réputation.
**RG 16**: Une recherche doit renvoyer une liste de résultats.
**RG 17**: Une recherche peut avoir des résultats mis en avant

## Modérateur
**RG 18**: Un modérateur est un membre
**RG 19**: Un modérateur doit avoir des droits de modération.

## Membre
**RG 20**: Un membre doit avoir un compte discord.
**RG 21**: Un membre peut commenter une ressource.
**RG 22**: Un membre peut noter positivement ou négativement une fois par solution/ressource
**RG 23**: Un membre peut demander de l'aide
**RG 24**: Un membre peut valider une réponse sur ses discussions d'aide

## Profil
**RG 25**: Un profil doit avoir:
- Un nom
- Un prénom
- Une présentation
**RG 26**: Un profil est unique à chaque membre
**RG 27**: Un profil peut avoir un compte GitHub lié
**RG 28**: Un profil peut avoir un compte LinkedIn associé
**RG 29**: Un profil doit être uniquement modifiable par l'auteur
**RG 30**: Un profil peut être supprimé par un modérateur ou l'auteur

## Étiquette
**RG 31**: Une étiquette doit avoir un nom.
**RG 32**: Une étiquette doit être unique.

## Question
**RG 33**: Une question doit avoir:
- Un titre de sujet
- Un corps
- Une date de publication
- Un auteur
**RG 34**: Une question peut être voté positivement ou négativement
**RG 35**: Une question peut avoir des étiquettes
**RG 36**: Une question doit référencer son message d'origine (même si mort)
**RG 37**: Une question peut être supprimée par l'auteur ou un modérateur

## Solution
**RG 38**: Une solution doit avoir:
- Un auteur
- Un corps
- Une date de publication
**RG 39**: Une solution doit apartenir à une discussion d'aide
**RG 40**: Une solution peut être votée positivement ou négativement
**RG 41**: Une solution peut être supprimée par l'auteur ou un modérateur
