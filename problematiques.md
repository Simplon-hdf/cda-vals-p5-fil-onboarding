
#  Problématiques, Problèmes et Solutions

Ce document recense les principales problématiques observées dans le serveur Discord de Simplon, notamment le manque de communication entre les membres, la faible participation transversale entre promotions, et la gestion peu efficace des ressources. Il propose des solutions concrètes pour améliorer la cohésion, favoriser le partage et dynamiser la communauté.

---

##  Table des Problématiques et Solutions

| **Problématique**                            | **Problème**                                                                                                                                          | **Solution**                                                                                                                                                                                                                                                                      |
|---------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **1.Manque de communication entre les membres du Discord** | **Manque de communication inter-promotions**                                                  | - **Introduction automatique du serveur** à l’arrivée pour présenter les espaces partagés et règles.<br>- **Système de profil public** pour chaque membre avec intérêts, compétences, promo, etc.<br>- **Recherche de membres** par critères pour encourager les connexions. |
|          | **Faible participation globale**                                                                                        | - **Gamification de la participation** : points, niveaux et badges pour les actions utiles (réponses, partages, feedbacks).<br>- Activités communautaires (défis, mini-jeux, quiz, sondages).                                                                                 |
|        |  **Manque de création de réseau**                                                                                    | - **Espaces d’entraide thématiques** (code, soft skills, etc.) ouverts à tous Communauté de Simplon.<br>- Forums de questions/réponses où chacun peut participer.<br>- Encouragement du mentorat inter-promotions.                                                             |
| **2. Partage de ressources confus et redondant** | **Les ressources sont mal organisées, partagées plusieurs fois et difficilement retrouvables.**                                                          | - **Centralisation des ressources** dans une zone commune, accessible à tous.<br>- **Bot de recherche** avec filtres par type, mot-clé, auteur, date, etc.<br>- **Système d’épinglage** pour mettre en avant les ressources importantes.                                    |
|  | **Absence de feedback sur les ressources**                  | - **Système de vote (👍👎)** sur chaque ressource pour favoriser le tri naturel par la communauté.<br>- Les ressources les mieux notées sont mises en avant automatiquement par le bot.<br>- Encouragement des commentaires pour contextualiser les votes.                     |


##  Conclusion

Avec ces solutions axées sur **l’interaction**, **l’automatisation**, et **la gamification**, nous espérons créer un environnement où chaque membre peut **participer activement**, **trouver rapidement ce dont il a besoin**, et **développer des relations inter-promotions durables**.
