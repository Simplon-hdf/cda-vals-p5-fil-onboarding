# RBAC - Community

## RESSOURCES

| Permission                                                | Membre | Modérateur |
|-----------------------------------------------------------|--------|------------|
| Accès aux ressources                                      | ✅     | ✅         |
| Créer une ressource                                       | ✅     | ✅         |
| Modifier ses propres ressources                           | ✅     | ✅         |
| Supprimer ses propres ressources                          | ✅     | ✅         |
| Modifier le titre de toutes les ressources                | ❌     | ✅         |
| Modifier les étiquettes discord de toutes les ressources  | ❌     | ✅         |
| Modifier les étiquettes bot de toutes les ressources      | ❌     | ❌         |
| Supprimer une ressource                                   | ❌     | ✅         |
| Voter une ressource comme utile / inutile                 | ✅     | ✅         |
| Ajouter des tags aux ressource                            | ✅     | ✅         |
