#  Exigences Fonctionnelles et Non Fonctionnelles

Une liste des exigences fonctionnelles et non fonctionnelles pour le projet **Onboarding** :

| 🛠️ Fonctionnelles                                                         | 🔐 Non fonctionnelles                                 |
| ------------------------------------------------------------------------- | ----------------------------------------------------- |
| **Création automatique des rôles, groupe, canaux, salons et forums pour une promo**   | **Aucune commande slash côté apprenant**  |
| **Template pour les promos**                                              | **Bonne ergonomie, fonctionnement intuitif**          |
| **Auto-assignation de rôles, puis validation par CDP**                    | **Manipulations gardées à un minimum**                |
| **Archivage des promos (qui, quoi, quand)**                               | **Respect du RGPD**                                   |
| **Création et archivage programmé des promos**                            | **Moindre privilège**                                 |
