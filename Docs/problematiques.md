# Problématiques du système d'onboarding Discord

Ce tableau synthétise les problèmes identifiés dans le questionnaire ainsi que les solutions proposées sous forme d’automatisation et d’amélioration de l’expérience utilisateur.

| **Problématique**                        | **Problème**                                                                                       | **Solution proposée**                                                                                     |
|-----------------------------------------|----------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| **Création manuelle de promos**         | La création manuelle des salons est longue                                                         | Automatisation de la création des salons à l’aide de templates réutilisables                              |
|                                         | La création manuelle des rôles est longue                                                          | Génération automatique des rôles lors de la création d’une promo                                           |
|                                         | L’attribution manuelle des rôles est fastidieuse                                                   | Attribution automatique ou via une commande /join_promo                                                   |
|                                         | Risques d’erreurs humaines dans la création                                                        | Utilisation de processus standardisés et scripts contrôlés                                                 |
|                                         | Complexité pour les utilisateurs non expérimentés sur Discord                                      | Système simplifié, interface intuitive via bot                                                             |
|                                         | Manque d’uniformité dans la structure des promos                                                   | Utilisation de conventions de nommage et modèles fixes                                                     |
|                                         | Difficulté à naviguer entre les promos actives                                                     | Séparation claire entre promos actives et archivées                                                       |
